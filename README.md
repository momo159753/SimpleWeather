# SimpleWeather

---
纯 Kotlin 开发的一款简单的天气应用。

国际惯例，先看截图：

![SimpleWeather][1]


## 关于这个应用
天气接口来源于 [Apixu][2]，UI 参考于琥珀天气。
使用 Kotlin 语言开发，网络请求采用 RxKotlin + Retrofit 实现。
其没有使用 mvp,mvvm,drgger 之类的框架，总体而言，代码与逻辑比较简单，更多的算是对 Kotlin 开发 Android 的初次尝试。
有什么问题，欢迎 issue 。
